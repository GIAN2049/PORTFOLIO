Proyectos Academicos

  1. Clinica Medijoven

    21, Mayo 2023 — 25, Junio 2023

    Este proyecto de software desarrollado en Spring Boot se centra en la gestión eficiente de una clínica médica. Proporciona una plataforma para llevar un registro detallado de medicamentos y sus tipos, pacientes, doctores, recetas médicas y citas médicas. Además, incluye un sistema de administración de usuarios con roles para garantizar la seguridad y el acceso adecuado a las diferentes partes del sistema.

  2. ANALISIS DE LICENCIA DE FUNCIONAMIENTO

    25, Septiembre 2022 - 29, Noviembre 2022

    Este proyecto de software, desarrollado con una variedad de tecnologías, tiene como objetivo simplificar y mejorar el proceso de análisis y gestión de licencias de funcionamiento para empresas en la Municipalidad de Miraflores, Lima. El sistema proporciona una plataforma eficiente para la solicitud, revisión y seguimiento de licencias, así como la generación de informes útiles para las autoridades municipales.

  3. SISTEMA DE GESTION DE MATRICULAS

    24, Abril 2022 - 20, Junio 2022

    Este proyecto de software desarrollado en Eclipse se enfoca en la gestión eficiente de matrículas académicas en una institución educativa. El sistema permite el registro de alumnos, asignaturas, docentes, periodos académicos, matrículas, y ofrece la capacidad de generar reportes detallados. Además, se incluye un sistema de inicio de sesión para garantizar la seguridad y el acceso adecuado a las funcionalidades.
